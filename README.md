# Setting up the project
1. Run `pip install requirements.txt` (or install the required packages via an IDE)
2. Extract the data folder `ECG_data` into the directory. The folder structure should like this

        ML_EE_ARR/
            ECG_data/
                beats_nsvfq_48p.csv
                test_data.csv
                train_data.csv
                val_data.csv
            main.py
            other files
